QA AI 도우미무료

ChatGPT를 피그마 안에서 사용하는 플러그인이에요.
아래 파일 3개를 받아서 피그마에 등록하세요.

📄 manifest.json플러그인 설정 파일 ⚙️ code.js플러그인 로직 파일 🖼️ ui.html플러그인 화면 파일

📋 설치 방법

  1. 위 파일 3개를 같은 폴더에 저장
  2. 피그마 열기 → 메뉴 → Plugins
  3. Development → Import plugin from manifest
  4. 저장한 폴더의 manifest.json 선택
  5. 설치 완료! 플러그인 실행 후 ChatGPT API 키 입력
