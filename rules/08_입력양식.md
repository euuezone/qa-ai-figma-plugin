# 입력양식

| 항목ID | 채널 | 입력구분 | 입력항목 | 입력내용 | 필수여부 | 작성예시 |
|---|---|---|---|---|---|---|
| I001 | 공통 | 기본정보 | 화면명 | TC 작성 대상 화면명 입력 | 필수 | 해외주식 투자정보 |
| I002 | 공통 | 기본정보 | 채널 | MTS, HTS, 홈페이지 중 해당 채널 입력 | 필수 | HTS |
| I003 | 공통 | 기본정보 | 화면성격 | 조회성, 처리성, 혼합형, 불명 중 선택 | 필수 | 조회성 |
| I004 | 공통 | 기본정보 | 업무구분 | 업무 도메인 또는 메뉴 성격 입력 | 권장 | 투자정보 |
| I005 | 공통 | 기본정보 | 화면단계 | 단일 화면 또는 단계형 화면 여부 입력 | 선택 | 1단계 정보 입력 |
| I006 | 공통 | 입력자료 | 화면 캡처본 | 실제 화면 구성 확인용 이미지 첨부 | 필수 | 화면 캡처 이미지 첨부 |
| I007 | 공통 | 입력자료 | 화면 설계서 | 화면 흐름, 버튼, 입력 항목, 팝업 확인용 설계서 이미지 첨부 | 선택 | 화면 설계서 이미지 첨부 |
| I008 | 공통 | 입력자료 | 도움말 페이지 | 기능 설명 확인용 도움말 이미지 또는 텍스트 첨부 | 선택 | 도움말 페이지 이미지 첨부 |
| I009 | 공통 | 입력자료 | 작성 가이드 | 고객사 또는 프로젝트 TC 작성 기준 첨부 | 선택 | TC 표기 규칙 가이드 첨부 |
| I010 | 공통 | 보충설명 | 화면 동작 설명 | 이미지만으로 확인 어려운 화면 동작 입력 | 권장 | 왼쪽 항목 선택 시 오른쪽 내용 변경 |
| I011 | 공통 | 보충설명 | 데이터 항목 설명 | 조회 결과 테이블 또는 목록의 컬럼명 입력 | 권장 | 일자, 제목, 작성자 |
| I012 | 공통 | 보충설명 | 버튼 동작 설명 | 버튼 선택 시 호출되는 화면, 팝업, 처리 결과 입력 | 권장 | [조회] 버튼 선택 시 결과 리스트 조회 |
| I013 | 공통 | 보충설명 | 팝업 설명 | 팝업 호출 조건과 팝업 목적 입력 | 선택 | 신청 전 확인 팝업 호출 |
| I014 | 공통 | 보충설명 | 완료 화면 설명 | 처리 완료 후 완료 화면 구성 입력 | 선택 | 신청 완료 후 계좌번호, 상품명, 신청금액 노출 |
| I015 | 공통 | 보충설명 | 예외 조건 설명 | 오류 메시지, 조건 미충족, 입력 제한 조건 입력 | 선택 | 필수값 미입력 시 오류 메시지 노출 |
| I016 | 공통 | 보충설명 | 작성 제외 요청 | 사용자가 작성하지 않길 원하는 항목 입력 | 선택 | 닫기 버튼, 스크롤, 단순 안내 문구 제외 |
| I017 | 공통 | 보충설명 | 작성 방식 요청 | 행 압축, 예상 결과 공란, 특정 표현 등 요청 입력 | 선택 | 버튼 예상 결과는 공란 처리 |
| I018 | 공통 | 보충설명 | 확인 불가 항목 | 이미지에서 식별되지 않는 항목 입력 | 선택 | 테이블 컬럼명 확인 불가 |
| I019 | MTS | 채널정보 | MTS 특이사항 | MTS 화면의 바텀시트, 단계형 플로우, 모바일 전용 동작 입력 | 선택 | 계좌 선택 시 바텀시트 호출 |
| I020 | HTS | 채널정보 | HTS 특이사항 | HTS 화면의 조회 조건, 결과 리스트, 좌우 구조, 차트 영역 설명 입력 | 선택 | 좌측 항목 선택 시 우측 상세 데이터 변경 |
| I021 | 홈페이지 | 채널정보 | 홈페이지 특이사항 | 홈페이지 화면의 문서 호출, 새 화면, 조회 결과 구조 입력 | 선택 | 약관 문서 선택 시 새 화면 호출 |
| I022 | 공통 | 출력요청 | 출력 컬럼 | 출력할 TC 컬럼 기준 입력 | 필수 | 화면구분1, 화면구분2, 테스트 구분, 테스트 시나리오, 예상 결과 |
| I023 | 공통 | 출력요청 | 출력 방식 | 표, TSV, CSV, 마크다운 등 출력 방식 입력 | 선택 | 구글시트 복붙용 TSV |
| I024 | 공통 | 출력요청 | 행 압축 여부 | 유사 TC 통합 여부 입력 | 선택 | 동일 예상 결과는 통합 |
| I025 | 공통 | 출력요청 | 검수 요청 | 생성 후 규칙 위반 검수 여부 입력 | 선택 | 작성규칙, 판단규칙, 제외규칙 기준 검수 |
| I026 | 공통 | 보안 | 민감정보 처리 | 실계좌, 고객명, 주민번호, 연락처 등 마스킹 여부 입력 | 필수 | 계좌번호 마스킹 처리 |
| I027 | 공통 | 보안 | 내부정보 처리 | 내부망 주소, 인증정보, 계정정보 등 제거 여부 입력 | 필수 | 내부 URL 제거 |
| I028 | 공통 | 검수정보 | 작성 기준 버전 | 적용할 규칙시트 버전 입력 | 권장 | TC_작성엔진_규칙시트_v0.1 |
| I029 | 공통 | 검수정보 | 담당자 메모 | 작성자 또는 검수자가 참고할 메모 입력 | 선택 | 화면 설계서 없음, 도움말 기준 작성 |
| I030 | 공통 | 검수정보 | 최종 확인 기준 | AI 생성 결과 검수 시 우선 확인할 기준 입력 | 선택 | 테스트 구분, 예상 결과 표현, 추정 작성 여부 확인 |
| I031 | 공통 | 입력방식 | 이미지 붙여넣기 | Figma 또는 클립보드에서 복사한 이미지를 앱 입력 영역에 직접 붙여넣기 | 필수 | Figma 이미지 복사 후 붙여넣기 |
| I032 | 공통 | 입력방식 | 다중 이미지 붙여넣기 | 화면 캡처, 도움말, 설계서 이미지를 여러 장 입력 | 필수 | 여러 이미지 붙여넣기 후 유형 선택 |
| I033 | 공통 | 입력방식 | 이미지 유형 선택 | 붙여넣은 이미지가 화면 캡처, 도움말, 설계서 중 무엇인지 선택 | 필수 | 이미지 유형을 화면 캡처로 선택 |
